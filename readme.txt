
Derleme kodu:
g++ main.c -lbgi -lgdi32 -lcomdlg32 -luuid -loleaut32 -lole32 -static-libgcc -static-libstdc++

Kullanım:
Hazır girdi ya da kendinizin oluşturduğu koordinat verilerinin bulunduğu dosyaları (.txt uzantılı dosya)
.exe dosyası ile aynı klasöre konulmalıdır.

Program başlangıcında .txt uzantılı dosyanın adı girilmelidir.

Örnek dosya veri girişi : 
-432 -100
300 6.3
150 256.3
241 -35.5
128 169

Format:
Noktanın x değeri Noktanın y değeri
Noktalar yeni satır ile ayrılmalıdır
